# Opposite number

---
### Instructions

Very simple, given an integer or a floating-point number, find its opposite.

Example:

```py
 1: -1
14: -14
-34: 34

```

---

### Solution

```py
def opposite(number):
    return -number
```

* Time Complexity: O(1)
* Space Complexity: O(1)
